#AI #NLP

**Natural Language Processing (NLP)** 是研究如何让计算机处理、理解和生成自然语言的领域。

NLP 是 Artificial Intelligence (AI) 的重要方向之一，现代 NLP 主要依赖 [Machine Learning](<./Machine%20Learning.md>) 和 [Deep Learning](<../Neural%20Networks/Deep%20Learning.md>) 方法。

## Core Tasks

- [Language Modeling](<../Language%20modeling/00%20-%20Maps%20and%20Overview/Language%20Modeling.md>)
- Text classification
- Machine translation
- Question answering
- Information extraction
- Text generation
- Dialogue systems

## 与 Language Modeling 的关系

[Language Modeling](<../Language%20modeling/00%20-%20Maps%20and%20Overview/Language%20Modeling.md>) 是现代 NLP 的核心问题之一，目标是建模 token sequence 的概率分布。

现代大语言模型通常先通过 language modeling objective 进行训练，再用于 text generation、question answering、dialogue 等任务。
## Modern NLP

现代 NLP 的核心模型通常是基于 [Transformer](<../Transformer/Transformer.md>) 的 language models，例如 GPT、BERT、T5 等。
